Returns: A JSON array of Pokémon objects.

Get Pokemon By ID

Method: GET, HEAD

Endpoints: /getPokemonById/{id}, /headPokemonById

Returns: A JSON object of the requested Pokémon.

Format: { id: number, name: string, img: string, type: [string], height: number, weight: number, weaknesses: [string], next_evolution?: [object] }
