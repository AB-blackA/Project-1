API Client Function Descriptions

Get Pokemon

This function retrieves a list of all Pokemon from the API.

Get Pokemon By ID

This function retrieves a specific Pokemon based on the provided ID.

Get Pokemon By Name

This function retrieves a specific Pokemon based on the provided name.

Get Pokemon By Height

This function retrieves a list of Pokemon based on the specified height.

Get Pokemon By Weight

This function retrieves a list of Pokemon based on the specified weight.

Get Pokemon By Type

This function retrieves a list of Pokemon based on the specified type.

Add Pokemon

This function allows you to add a new Pokemon to the API with the specified attributes, including ID, Name, Image, Type(s), Height, Weight, and optional Evolutions.

Delete Pokemon

This function allows you to delete a Pokemon from the API based on the provided ID.
